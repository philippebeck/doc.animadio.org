Variables

Why CSS variables ?

The only reason Animadio is fully written in CSS is the presence of custom properties & CSS variables.

Custom properties have the particularity of being overloaded directly from the custom stylesheet file...

It is therefore possible to completely customize any element of the framework thanks to the CSS variables, in order to adapt it to each use case !

How to use them ?

Just choose the required variable with its selector in the following file :

Variables

Then in your custom stylesheet file, write a CSS declaration with the necessary selector, the property to modify & the value you want !
